三個資料集下載與使用指南

OTRF Security-Datasets

Windows Sysmon + 安全事件日誌，含精確 ATT&CK 子技術標籤

Windows EventLog 有 GT 標籤 約 1–5 MB / 資料集
下載
# 需要先安裝 Git（git-scm.com）
git clone https://github.com/OTRF/Security-Datasets.git
pip install python-evtx
或用腳本自動下載並轉換
python download_otrf.py
手動轉換（若已 clone）
python "Convert datasets to jsonl.py" --source evtx
  --input .\Security-Datasets\datasets\atomic\windows
  --output otrf_hunting.jsonl
建行為鏈並跑 pipeline
python aggregate_chains.py --input otrf_hunting.jsonl --output chains.jsonl
python universal_soc_pipeline.py

UWF-ZeekData24

Zeek 網路流量，真實非模擬封包，含 ATT&CK 標籤

Zeek CSV 有 GT 標籤 每個 CSV 約 200–300 MB
下載（瀏覽器直接點擊）
瀏覽器開啟：https://datasets.uwf.edu/data/
找到 UWF-ZeekData24 資料夾
下載任一 part-xxxxx.csv 檔案
轉換
python convert_uwf.py
  --input .\part-00000-xxxxx.csv
  --output uwf_hunting.jsonl --limit 2000
建行為鏈並跑 pipeline
python aggregate_chains.py --input uwf_hunting.jsonl --output chains.jsonl
python universal_soc_pipeline.py

EVTX-to-MITRE-Attack

270+ 個 Windows EVTX 情境，資料夾名稱即 TTP ID

Windows EVTX 有 GT 標籤 約 200 MB
下載
git clone https://github.com/mdecrevoisier/EVTX-to-MITRE-Attack.git
pip install python-evtx
轉換（自動從資料夾名稱萃取 TTP）
python "Convert datasets to jsonl.py" --source evtx
  --input .\EVTX-to-MITRE-Attack
  --output evtx_hunting.jsonl

# 過濾掉無標籤筆數
python -c " import json; lines=[l for l in open('evtx_hunting.jsonl',encoding='utf-8') if json.loads(l)['valid_ttps']!=['Unknown Threat']]; open('evtx_labeled.jsonl','w',encoding='utf-8').writelines(lines); print(len(lines),'筆')"
建行為鏈並跑 pipeline
python aggregate_chains.py --input evtx_labeled.jsonl --output chains.jsonl
python universal_soc_pipeline.py

使用真實 LOG（無標準答案）

Hunting Mode（實戰狩獵模式）

只要 JSONL 裡的每筆資料沒有 valid_ttps 欄位，系統就自動進入 Hunting Mode，不需要任何額外設定。

從 Windows 事件檢視器匯出真實 LOG
# 在 Windows 事件檢視器中：
# 1. 右鍵 Security / System / Application 日誌
# 2. 選「另存所有事件」→ 儲存為 .evtx
# 3. 用轉換腳本處理（不帶 valid_ttps 欄位）

python "Convert datasets to jsonl.py" --source evtx
  --input .\my_real_logs.evtx --output real_hunting.jsonl
或手動建立 JSONL（適合單筆測試）
# 每行一個 JSON，不加 valid_ttps 欄位
{"id": "REAL_001", "text": "EventID=4688 Host=mypc User=alice CommandLine=powershell -enc ...", "timestamp": "2026-01-01T10:00:00Z"}
{"id": "REAL_002", "text": "EventID=4698 Host=mypc User=alice CommandLine=schtasks /create ...", "timestamp": "2026-01-01T10:02:00Z"}
執行 Hunting Mode
# 建行為鏈（系統偵測沒有 valid_ttps → 自動 Hunting Mode）
python aggregate_chains.py --input real_hunting.jsonl --output real_chains.jsonl
python universal_soc_pipeline.py
